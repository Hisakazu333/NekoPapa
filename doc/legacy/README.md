# 历史文档

本目录保存已经退出 NekoPapa 当前产品口径的材料。它们可以解释早期设想，但不能作为实现状态、架构边界、产品命名或发布承诺。

## 使用规则

- 新 Issue 和 Pull Request 不得引用本目录证明功能已经批准或实现
- 与当前 README、`doc/product/` 或 `doc/architecture/` 冲突时，以当前规范和代码证据为准
- 历史文档保留原始品牌与措辞，避免伪造项目历史
- 有价值的产品概念必须重新进入 GitHub Issue，并补齐范围、owner、状态模型和验收条件
